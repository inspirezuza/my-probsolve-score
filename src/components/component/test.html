    ในโจทย์บางข้อ ถ้าข้อมูลนำเข้ามีขนาดใหญ่มาก การใช้ cin เฉย ๆ จะอ่านข้อมูลไม่ทัน ให้เพิ่มบรรทัดนี้ด้านหน้าก่อนเรียกใช้ cin ครั้งแรก เช่นใส่ในบรรทัดแรกของ main

    ios::sync_with_stdio(false);
cin.tie(NULL);

    จะทำให้อ่านเร็วขึ้น และอย่าใช้ endl ให้ใช้ "\n" แทน เช่น

    cout << x << "\n";

    การอ่าน EOF (ข้อ wavio & coin change) อ่านด้วย cin, อ่านด้วย scanf

    ข้อ UVa 11462 - Age sort: ควรทำอย่างที่ระบุด้านบนด้วย นอกจากนี้ตอนพิมพ์ผลลัพธ์อย่าให้มีช่องว่างสุดท้ายเกินมา อาจจะตรวจแล้วไม่ผ่านเนื่องจาก presentation error
